# Kongko Spa | Premium Men's Spa Landing Page

A premium men's spa landing page built with React, Tailwind CSS, and Framer Motion.

## Features
- **Smoked-Gold Glass Design**: Custom overlay reflections, light-refracting borders, and premium gold accents.
- **Ken Burns Transitions**: Full-bleed responsive viewport background images with slow Ken Burns scaling.
- **Staggered Animations**: Word-by-word paragraph text animations triggered dynamically on visibility.
- **Digital Menu & Booklet**: Displaying pricing, categories, and signature offerings.
- **Smooth Navigation**: Smooth scrolling single-page anchor jumps.
